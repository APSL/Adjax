Where to get Adjax

You can download version 1.0 here: adjax-1.0.0.tar.gz, or install using `easy_install adjax`

SVN users can hook into the current development version here:

svn co http://django-adjax.googlecode.com/svn/trunk/adjax

Or you can browse the source code online and submit bug reports

Adjax is released under an open-source (New BSD) licence. It is free to use, modify and redistribute, and contributions are both welcome and encouraged.
